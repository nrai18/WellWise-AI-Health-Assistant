## Personalized Planning Modules ğŸğŸ‹ï¸ğŸ’§

The platform provides dedicated modules for comprehensive wellness management:

### 1. Diet Planner & Customization
* **Personalized Diet Plans:** Generates daily meal plans based on **age, gender, height, weight, activity level, primary goal (e.g., Maintain weight)**, dietary preference, and number of meals per day.
* **Custom Plan Creator:** Allows granular control over meal planning by enabling users to set **maximum nutritional targets** for **Calories, Protein, Carbs, and Fat**. Users can also refine their search by specifying **Dietary Preference** and **Cuisine/Ethnicity**.

### 2. Exercise Hub
* **Weekly Workout Plan Generator:** Creates a structured weekly exercise plan based on:
    * **Fitness Level** (e.g., Beginner).
    * **Primary Goal** (e.g., Weight Loss).
    * **Available Equipment** (Bodyweight Only, Dumbbells, Resistance Bands, Full Gym).
    * Configurable **Workout Days per Week** and **Time per Session (minutes)**.

### 3. Hydration Hub
* **Personalized Hydration Guidance:** Provides recommendations and tracking to help users meet their daily water intake goals.
